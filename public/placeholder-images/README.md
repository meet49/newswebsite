# Placeholder Images

This directory contains placeholder images for the news website.

## Usage

Replace these placeholder images with your actual news images:

- `news-placeholder.jpg` - Default placeholder for articles without images
- Add more placeholder images as needed

## Image Guidelines

For best results:
- **Dimensions**: 800x600px minimum (4:3 aspect ratio)
- **Format**: JPG or PNG
- **Size**: Keep under 200KB for optimal loading
- **Quality**: High resolution but optimized for web
